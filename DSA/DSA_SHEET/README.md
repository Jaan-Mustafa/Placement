## How to use this
1. Go top to bottom — steps build on each other.
2. Open a step's `README.md`, solve a problem, tick the box.
3. Save your solution as `<problem_name>.cpp` (or `.java`) in that step's folder.
4. Watch the linked Striver video/editorial if stuck — every problem on takeUforward has one.
